************
Compilation
************
gcc -Wall -ansi -pedantic -g src/binarytree.c src/main.c -o main -I include

***********************
Running the program(s)
***********************
./main .
For file manager commands,
1 means print all file path
2 means delete
3 means rename
4 means move
5 means exit the program


*****************
Known Limitations
*****************
The program can find at most 256 files or directories under the target directory
